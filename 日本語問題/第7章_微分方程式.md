# 第7章 微分方程式の数値解法（シミュレーションの基礎）

第6章までで、MATLABの基本的な文法と行列計算を習得しました。第7章では、これまでの知識（関数、ベクトル、グラフ）を総動員して、力学系やロボットの動きをシミュレーションするための「常微分方程式（ODE: Ordinary Differential Equation）の数値解法」を学びます。

MATLABはシミュレーションに非常に強く、複雑な微分方程式も組み込み関数を使って簡単に解くことができます。

## 1. 無名関数（Anonymous Functions）
微分方程式を解く前に、関数をより簡単に定義する方法を学びます。第5章では別ファイル（Mファイル）として関数を作成しましたが、数式が短い場合は、1行で完結する「無名関数」を使うと便利です。

```matlab
% xを2乗する無名関数を定義し、変数 f に格納する
f = @(x) x^2;

% 関数の実行
y = f(5); % y には 25 が代入されます

% 2つの入力を持つ無名関数
g = @(x, y) x^2 + y^2;
z = g(3, 4); % z には 25 が代入されます
```

### 構文の解説
* `@(入力変数)` : これから関数を定義するという宣言です。アットマーク `@` の直後のカッコ内に、関数が受け取る引数を指定します。その後ろに計算式を記述します。わざわざ別ファイルを作成しなくて済むため、微分方程式の定義などに多用されます。

## 2. ode45 を使った微分方程式の解法
MATLABで最も標準的に使われる常微分方程式のソルバー（解を求める関数）が `ode45` です。ルンゲ・クッタ法という精度の高い数値計算アルゴリズムを用いています。

`ode45` を使うには、解きたい微分方程式を $\frac{dy}{dt} = f(t, y)$ という「1階の微分方程式（状態方程式）」の形に変形し、その右辺 $f(t, y)$ を関数として定義します。

```matlab
% 例： dy/dt = -2y という微分方程式を解く

% 1. 微分方程式の右辺を無名関数として定義
% ode45で使う関数は、必ず (t, y) の順で引数を取る必要があります
dydt = @(t, y) -2 * y;

% 2. シミュレーションする時間の範囲（開始時刻と終了時刻） [s]
tspan = [0, 5];

% 3. 初期条件（t=0 のときの y の値）
y0 = 10;

% 4. ode45 の実行
% 戻り値として、計算された時間ベクトル t と、その時の状態ベクトル y が得られます
[t, y] = ode45(dydt, tspan, y0);

% 5. 結果のプロット
plot(t, y);
xlabel('Time t [s]');
ylabel('State y');
grid on;
```

### 関数の解説
* `ode45(関数, 時間範囲, 初期値)` : 与えられた初期値からスタートし、指定された時間範囲にわたって微小な時間ステップごとに変化量を計算し、次の状態を予測していく関数です。時間ステップの幅はMATLABが自動で最適化してくれます。

## 例題：空気抵抗を受けるドローンの落下速度シミュレーション
質量 $m = 1.2$ [kg] のドローンが、モーターを停止して落下している状態を考えます。
重力による下向きの力 $mg$ に加え、速度 $v$ [m/s] に比例する空気抵抗 $-cv$ が上向きに働くとすると、運動方程式は以下のようになります。

$$m \frac{dv}{dt} = mg - cv$$

これを $\frac{dv}{dt} = \dots$ の形に変形すると、以下のようになります。

$$\frac{dv}{dt} = g - \frac{c}{m}v$$

重力加速度 $g = 9.8$ [m/s^2]、空気抵抗係数 $c = 0.5$ [N$\cdot$s/m] とし、初期速度 $v(0) = 0$ [m/s] として、$0$秒から $10$秒までの速度 $v$ の変化をシミュレーションし、グラフにプロットするコードを作成してください。

## 例題の解答

```matlab
% 物理パラメータの定義
m = 1.2;  % 質量 [kg]
g = 9.8;  % 重力加速度 [m/s^2]
c = 0.5;  % 空気抵抗係数 [N*s/m]

% 微分方程式の右辺 dv/dt = f(t, v) を無名関数で定義
% ※式の中に t は登場しませんが、ode45 の仕様として引数 (t, v) は必須です
dvdt = @(t, v) g - (c / m) * v;

% シミュレーションの条件設定
tspan = [0, 10]; % 0秒から10秒まで
v0 = 0;          % 初期速度 0 [m/s]

% ode45 による数値解の計算
[t, v] = ode45(dvdt, tspan, v0);

% 結果の可視化
plot(t, v, 'LineWidth', 2); % 線の太さを2にして見やすくする
xlabel('Time [s]');
ylabel('Velocity v [m/s]');
title('Drone Free Fall with Air Resistance');
grid on;
```

### 解答コードの解説
* `dvdt = @(t, v) g - (c / m) * v;` : 運動方程式から導いた $\frac{dv}{dt}$ の右辺を無名関数にしています。事前に定義した変数 `g`, `c`, `m` をそのまま関数内で使うことができます。
* このプログラムを実行すると、最初は速度が急激に増加しますが、空気抵抗が大きくなるにつれて加速度が減少し、やがて一定の速度（終端速度）に収束していく様子がグラフで確認できます。ロボティクスにおけるダイナミクスの解析は、この `ode45` を基礎として発展していきます。
