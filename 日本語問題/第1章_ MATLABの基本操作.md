# 第1章 MATLABの基本操作と変数の定義

ようこそ！このコースでは、MATLABの基礎の基礎から1章ずつ丁寧に解説していきます。第1章では「MATLABでの基本的な計算」と「変数の使い方」を学びます。

## 1. MATLABのコマンドウィンドウ
MATLABの「コマンドウィンドウ」は、ターミナルのように対話型でコマンドを打ち込む場所です。簡単な数式を入力すると、電卓のようにすぐに結果が返ってきます。

```matlab
% 足し算
3 + 5

% 掛け算とべき乗
2 * 4^2
```

### コードの解説
* `%` : **コメントアウト**を意味します。`%` 以降の文字はプログラムとして評価されず、メモとして扱われます。学習中のコードには積極的にメモを残しましょう。
* `+`, `*`, `^` : それぞれ加算、乗算、べき乗（この場合は $4^2$ で16）を行う算術演算子です。

## 2. 変数の定義とセミコロンの役割
計算結果を後で再利用するために「変数」にデータを格納します。MATLABでは変数の型（整数、小数など）を事前に宣言する必要はありません。

```matlab
% 変数aに数値を代入
a = 10;

% 変数bに数値を代入
b = 5;

% aとbを使って新しい変数cを計算
c = a * b;

% 結果を表示する
disp(c)
```

### コードと関数の解説
* `a = 10;` : 右辺の `10` を左辺の変数 `a` に**代入**します。
* `;` (セミコロン) : 行の末尾にセミコロンをつけると、**計算結果の画面への出力を省略（非表示）**します。変数の計算や格納はバックグラウンドで確実に行われます。MATLABでは、画面が途中計算の出力で埋め尽くされるのを防ぐために、行末に `;` をつけるのが基本的な作法です。
* `disp()` : **Display関数**です。指定した変数の内容や文字列をコマンドウィンドウに出力します。ここでは変数 `c` の中身である `50` を表示させています。

## 例題：運動方程式の計算
質量 $m = 2.5$ [kg] の物体に、加速度 $a = 9.8$ [m/s^2] が生じているとき、この物体に働いている力 $F$ [N] を計算するMATLABコードを作成してください。
（ヒント：力は質量と加速度の積 $F = ma$ で求まります。変数 `m` と `a` を定義してから `F` を計算してみましょう。）

## 例題の解答

```matlab
% 質量の定義 [kg]
m = 2.5;

% 加速度の定義 [m/s^2]
a = 9.8;

% 力の計算 (F = ma)
F = m * a;

% 結果の出力
disp('物体に働いている力 F [N] は以下の通りです:');
disp(F);
```

### 解答コードの解説
* `m` と `a` という変数にそれぞれ数値を代入し、行末に `;` をつけて不要な出力を防いでいます。
* `F = m * a;` で計算を行い、結果を変数 `F` に格納します。
* `disp('文字列')` のように、シングルクォーテーション `' '` で囲むと、そのテキストをそのまま画面に表示することができます。その直後に `disp(F)` を呼ぶことで、計算結果の数値を分かりやすく出力しています。
