# 第5章 関数とスクリプト（Mファイル）

第4章までで、MATLABの基本的な構文（計算、ベクトル、グラフ、制御構文）をマスターしました。しかし、これまでのようにコマンドウィンドウに直接命令を打ち込む方法では、複雑なプログラムを何度も実行したり、他の人と共有したりするのが大変です。

第5章では、コードを「ファイル」として保存し、再利用可能な「関数」を作る方法を学びます。MATLABでは、これらのファイルは拡張子が `.m` であるため、「Mファイル」と呼ばれます。

## 1. スクリプト（Script）
スクリプトは、コマンドウィンドウに打ち込む一連の命令を単に書き並べたテキストファイルです。
* **作成と実行**: 保存したファイル名（例：`myscript.m`）をコマンドウィンドウで入力するか、エディタの実行ボタンを押すだけで、中身が上から順に実行されます。
* **Neovimの利用**: 保存した情報を踏まえ、ターミナル環境であれば `nvim myscript.m` と打ち込んで編集し、MATLAB上で実行するという流れがスムーズです。

## 2. 関数（Function）
関数は、特定の「入力（引数）」を受け取り、処理を行って「出力（戻り値）」を返す仕組みです。スクリプトとの大きな違いは、**変数スコープの独立性**です。関数の中で使われる変数は、関数の外（ワークスペース）には影響を与えません。

### 基本的な関数の書き方
関数の作成には決まった型があります。

```matlab
function [result] = my_square(x)
    % MY_SQUARE 入力された数値を2乗して返す関数
    % 入力: x (数値)
    % 出力: result (xの2乗)

    result = x^2;
end
```

### コードの解説
* `function` : 「これは関数の定義です」と宣言するキーワードです。
* `[result]` : 出力される変数の名前です。複数を `[out1, out2]` のように指定することもできます。
* `my_square` : **関数名**です。ファイル名はこの名前と同じ（`my_square.m`）にする必要があります。
* `(x)` : 入力される変数の名前（引数）です。
* `end` : 関数の定義の終わりを示します。

## 3. 複数の入出力を持つ関数
工学計算では、複数のパラメータを渡して複数の結果を得ることがよくあります。

```matlab
function [area, circumference] = circle_calc(radius)
    % 円の面積と円周を計算する関数
    
    area = pi * radius^2;           % 面積 S = πr^2
    circumference = 2 * pi * radius; % 円周 L = 2πr
end
```

この関数を呼び出すときは、コマンドウィンドウで以下のように記述します。
`[a, c] = circle_calc(5);`

## 例題：運動エネルギーの計算関数の作成
質量 $m$ [kg] と速度 $v$ [m/s] を入力すると、物体の運動エネルギー $K$ [J] を計算して返す関数 `calc_energy` を作成してください。
運動エネルギーの公式は $K = \frac{1}{2} m v^2$ です。

作成後、その関数を呼び出して、質量 $80$ [kg]、速度 $20$ [m/s] の時のエネルギーを求めて表示するコードも書いてください。

## 例題の解答

### 1. 関数ファイル (`calc_energy.m` として保存)
```matlab
function [K] = calc_energy(m, v)
    % CALC_ENERGY 質量mと速度vから運動エネルギーKを計算する
    % 入力: m (質量), v (速度)
    % 出力: K (運動エネルギー)

    % 要素ごとの演算 .^ を使うと、速度がベクトルの場合にも対応できます
    K = 0.5 * m * v.^2;
end
```

### 2. 実行用コード（スクリプトまたはコマンドウィンドウ）
```matlab
% パラメータの定義
mass = 80;
velocity = 20;

% 自作した関数を呼び出す
energy = calc_energy(mass, velocity);

% 結果を表示
disp('計算された運動エネルギー [J]:');
disp(energy);
```

### 解答コードの解説
* `function [K] = calc_energy(m, v)` : 入力が2つ（`m`, `v`）、出力が1つ（`K`）の関数を定義しています。
* `v.^2` : 第2章で学んだドット演算子を関数内でも使うことで、この関数は「速度のリスト（ベクトル）」に対しても一括でエネルギーを計算できる汎用性の高いものになります。
* 呼び出し側では、関数で定義した `m`, `v` という名前ではなく、自分の好きな変数名（`mass`, `velocity`）を引数として渡すことができます。
