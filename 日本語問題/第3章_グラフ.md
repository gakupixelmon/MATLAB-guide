# 第3章 データの可視化（グラフの作成）

第2章のベクトル演算、お見事でした！MATLABの強力な計算能力を実感できたかと思います。第3章では、計算したデータを視覚的に理解するための「グラフの描き方」を学びます。MATLABは、数行のコードで非常に美しいグラフを作成できるのが特徴です。

## 1. 基本的なプロット（plot関数）
グラフを描くための最も基本的な関数が `plot` 関数です。横軸のデータ（ベクトル）と縦軸のデータ（ベクトル）を渡すだけで、自動的に折れ線グラフを描画してくれます。

```matlab
% xのデータを0から0.1刻みで2*pi（約6.28）まで作成
% pi は円周率 π を表すMATLABの組み込み変数です
x = 0:0.1:2*pi;

% yのデータとして、xの正弦（サイン）を計算
y = sin(x);

% xとyをプロット
plot(x, y)
```

## 2. グラフの装飾（ラベル、タイトル、グリッド）
グラフを描いただけでは、縦軸や横軸が何を表しているのか他の人には分かりません。関数を使ってグラフに情報を追加しましょう。

```matlab
plot(x, y)

% x軸とy軸に名前（ラベル）をつける
xlabel('時間 t [s]');
ylabel('振幅 y [m]');

% グラフにタイトルをつける
title('正弦波のグラフ');

% グラフにグリッド（格子線）を表示して見やすくする
grid on;
```
これらの装飾関数は、`plot` 関数の「後」に書くのがルールです。

## 3. 複数のグラフを重ねて描く（hold on / hold off）
通常、`plot` 関数を続けて2回実行すると、前のグラフは消えて新しいグラフに上書きされてしまいます。複数のグラフを同じ画面に重ねて表示したい場合は、`hold on` というコマンドを使います。

```matlab
y1 = sin(x);
y2 = cos(x);

% 1つ目のグラフを描画
plot(x, y1)

% グラフの保持を有効にする（上書きを防止）
hold on;

% 2つ目のグラフを描画（色は自動的に変わります）
plot(x, y2)

% 凡例（どれがどの線かの説明）を表示
legend('sin(x)', 'cos(x)');

% 保持を解除する（次のplotで画面がリセットされるようにする）
hold off;
```

## 例題：斜方投射（ボールの放物線）のグラフ化
初速度 $v_0 = 15$ [m/s] でボールを真上に投げ上げたときの、時間 $t$ [s] に対する高さ $y$ [m] の変化をグラフにしてください。（重力加速度 $g = 9.8$ [m/s^2] とします）

以下の条件を満たすMATLABコードを作成してください。
1. 時間 $t$ を `0` から `0.1` 刻みで `3` まで作成する。
2. 高さ $y$ を、公式 $y = v_0 t - \frac{1}{2} g t^2$ を用いて計算する。（※第2章で学んだ要素ごとの演算 `.^` や `.*` に注意！）
3. 縦軸と横軸を `plot` 関数でグラフ化する。
4. 横軸のラベルを `'Time [s]'`、縦軸のラベルを `'Height [m]'`、タイトルを `'Ball Trajectory'` とし、グリッド（`grid on`）を表示する。

## 例題の解答

```matlab
% パラメータの定義
v0 = 15;  % 初速度 [m/s]
g = 9.8;  % 重力加速度 [m/s^2]

% 時間tのベクトルを作成（0から0.1刻みで3まで）
t = 0:0.1:3;

% 高さyの計算
% tはベクトルなので、2乗には .^ を使用します
y = v0 * t - 0.5 * g * t.^2;

% グラフの描画
plot(t, y)

% グラフの装飾
xlabel('Time [s]');
ylabel('Height [m]');
title('Ball Trajectory');
grid on;
```

### 解答コードの解説
* `y = v0 * t - 0.5 * g * t.^2;` について：`t` がベクトルの場合、スカラー（単一の数字）との掛け算にはピリオドは不要ですが、ベクトル同士の掛け算やべき乗にはピリオドが必要です。ここでは `t.^2` がポイントになります。
* （補足）厳密には `v0 * t` は「スカラーとベクトルの掛け算」なので `.*` は不要ですが、`v0 .* t` と書いても正しく動作します。
* プロット後に装飾関数を連続して呼び出すことで、プロフェッショナルな見た目のグラフが完成します。
