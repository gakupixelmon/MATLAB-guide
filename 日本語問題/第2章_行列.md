# 第2章 ベクトルと行列の作成

第1章、お疲れ様でした！MATLABは「Matrix Laboratory（行列の研究室）」の略であり、複数のデータをまとめた「ベクトル」や「行列」の扱いが非常に得意な言語です。第2章では、複数のデータを一度に扱う方法を学びます。

## 1. ベクトル（1次元配列）の作り方
数値をカギカッコ `[ ]` で囲むことで、複数の数値を持ったベクトルを作成できます。

```matlab
% 行ベクトル（横に並ぶベクトル）
% スペースまたはカンマで区切ります
v_row = [1 2 3 4 5];

% 列ベクトル（縦に並ぶベクトル）
% セミコロンで区切ります
v_col = [1; 2; 3; 4; 5];
```

## 2. 連続したデータの一括作成
グラフを描いたりシミュレーションをしたりする際は、「0から10まで1刻みのデータ」などがよく必要になります。コロン `:` を使うとこれを一瞬で作成できます。

```matlab
% 構文: [開始値 : 増分 : 終了値]
% 0から2刻みで10までのベクトルを作成
t = 0:2:10; 
% 結果は [0, 2, 4, 6, 8, 10] となります

% 増分が1の場合は省略可能です
x = 1:5;
% 結果は [1, 2, 3, 4, 5] となります
```

## 3. 配列に対する計算（要素ごとの演算）
ここがMATLABの最も重要なポイントの一つです。ベクトル全体に対して掛け算やべき乗を行うときは、演算子の前にピリオド `.` をつけます。これを「要素ごとの演算」と呼びます。

```matlab
a = [1 2 3];
b = [4 5 6];

% 要素ごとの掛け算（1*4, 2*5, 3*6）
c = a .* b; 

% 要素ごとのべき乗（1^2, 2^2, 3^2）
d = a .^ 2; 

% ※足し算と引き算、スカラー（単一の数字）との掛け算にはピリオドは不要です
e = a + b;
f = 2 * a; % [2, 4, 6] になる
```
ピリオドを忘れると、数学的な「行列の積」として計算しようとしてしまい、エラーになることが多いので注意しましょう。

## 例題：自由落下の位置計算
ある物体を初速0で落下させます（重力加速度 $g = 9.8$ [m/s^2]）。
以下の条件を満たすMATLABコードを作成してください。

1. 時間 $t$ を `0` から `0.5` 刻みで `3` まで作成する。
2. 落下距離 $y$ を、公式 $y = \frac{1}{2} g t^2$ を用いて計算する。（※ $t$ はベクトルなので、要素ごとの演算に注意してください）
3. 計算結果 $y$ を表示する。

## 例題の解答

```matlab
% 重力加速度の定義 [m/s^2]
g = 9.8;

% 時間tのベクトルを作成（0から0.5刻みで3まで）
t = 0:0.5:3;

% 落下距離yの計算
% tはベクトルなので、べき乗には .^ を使用します
y = 0.5 * g * t.^2;

% 結果の出力
disp('各時間における落下距離 y [m] は以下の通りです:');
disp(y);
```

### 解答コードの解説
* `t = 0:0.5:3;` によって、`[0, 0.5, 1.0, 1.5, 2.0, 2.5, 3.0]` という7つのデータを持つベクトルが一瞬で作成されます。
* `t.^2` とピリオド付きのべき乗を使うことで、`t` の中身の7つの数字すべてに対して、ループ文などを使わずに一度に2乗の計算を行っています。これがMATLABの強力な機能です。
