# 第4章 条件分岐と繰り返し処理（if文とfor文）

第3章まで順調ですね！ここまでで、MATLABを使って一連の計算を行い、グラフを描くことができるようになりました。第4章では、プログラムに「条件に応じた判断」と「同じ処理の反復」を行わせるための制御構文を学びます。

## 1. if文による条件分岐
ある条件が満たされたときだけ特定の処理を行いたい場合は、`if` 文を使用します。MATLABでは、ブロックの終わりに必ず `end` を書くルールがあります。

```matlab
x = 10;

% xが5より大きいかどうかの判定
if x > 5
    disp('xは5より大きいです');
elseif x == 5
    disp('xは5です');
else
    disp('xは5より小さいです');
end
```

### 比較演算子の解説
条件式で使う記号（比較演算子）は以下の通りです。
* `==` : 等しい（代入の `=` とは異なり、2つ並べることに注意してください）
* `~=` : 等しくない（他の言語では `!=` と書くことが多いですが、MATLABでは `~=` です）
* `>`, `<` : より大きい、より小さい
* `>=`, `<=` : 以上、以下

## 2. for文による繰り返し処理
同じ処理を何度も繰り返したい場合は、`for` 文を使用します。ベクトルを使って「どこからどこまで繰り返すか」を指定します。

```matlab
% 1から5まで、変数iを1ずつ増やしながら処理を繰り返す
for i = 1:5
    % 繰り返したい処理
    disp(i^2); 
end
```

### コードの解説
* `i = 1:5` : 第2章で学んだコロン演算子です。`[1, 2, 3, 4, 5]` というベクトルが作られ、変数 `i` にはループが回るごとに1、2、3...と順番に数値が代入されていきます。
* `end` : `for` ループの終わりを示します。MATLABでは字下げ（インデント）は必須ではありませんが、コードを読みやすくするために字下げを行うのが一般的です。

## 3. ベクトルの要素へのアクセス
`for` 文とベクトルを組み合わせることで、データの処理が柔軟になります。ベクトルの特定の場所にあるデータを取り出したり書き換えたりするには、丸括弧 `()` を使います。

```matlab
v = [10, 20, 30, 40];

% 3番目の要素を取り出して表示する
disp( v(3) ); % 30が表示されます

% 2番目の要素を書き換える
v(2) = 99; 
% v は [10, 99, 30, 40] になります
```
**※重要：** MATLABのインデックス（何番目かを表す数字）は、0ではなく **1から始まります**。

## 例題：センサーデータのノイズ除去
ロボットの距離センサーから取得した5つのデータがベクトルとして与えられています。
`distances = [1.5, 0.8, -0.2, 2.1, -0.5];`

距離がマイナスになることは物理的におかしいので、これはエラー（ノイズ）と考えられます。以下の条件を満たすMATLABコードを作成してください。

1. `for` 文を使って、`1` から `5` までループを回す（変数には $i$ などを使いましょう）。
2. ループの中で `if` 文を使い、$i$ 番目のデータ `distances(i)` が $0$ より小さいかどうかを判定する。
3. もし $0$ より小さければ、その値を $0$ に書き換える。
4. 最後に、修正された `distances` ベクトルを表示する。

## 例題の解答

```matlab
% センサーのデータ
distances = [1.5, 0.8, -0.2, 2.1, -0.5];

% 1番目から5番目まで順番に確認する
for i = 1:5
    % もし値が0より小さければ
    if distances(i) < 0
        % その値を0に書き換える
        distances(i) = 0;
    end
end

% 結果の出力
disp('修正後のデータ:');
disp(distances);
```

### 解答コードの解説
* `for` と `if` の両方のブロックを閉じるために、`end` が2つ必要になります。それぞれどのブロックを閉じているのか分かりやすくするため、字下げ（インデント）を整えています。
* `distances(i)` で要素を読み取り、マイナスであれば再度 `distances(i) = 0` で書き込みを行っています。これによって、ベクトルの中身が部分的に更新されます。

（補足：MATLABの実践的なプログラミングでは、このような処理は第2章で触れたような「ベクトル演算（論理インデックス）」を使って `distances(distances < 0) = 0;` のように1行で書くことが推奨されますが、まずは基礎となる `for` と `if` のロジックを理解することが非常に重要です。）
