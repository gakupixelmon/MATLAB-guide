# 第6章 行列の演算と連立方程式の解法

MATLAB（Matrix Laboratory）の真骨頂は、その名の通り「行列」の扱いにあります。ロボット工学やドローンの姿勢制御、高度なシミュレーションにおいて、行列演算は必須の数学ツールとなります。第6章では、2次元の行列の扱いと、連立方程式を圧倒的な速度で解く方法を学びます。

## 1. 2次元配列（行列）の作成
第2章で学んだベクトルを拡張し、行と列を持つ行列を作成します。行を変える（改行する）には、セミコロン `;` を使います。

```matlab
% 2行3列の行列 A を作成
% スペースまたはカンマで列を区切り、セミコロンで行を区切ります
A = [1 2 3; 4 5 6];

% 3行3列の正方行列 B を作成
B = [1 2 3; 4 5 6; 7 8 9];
```

## 2. 行列の演算（積、転置、逆行列）
第2章で「要素ごとの演算（`.` ピリオドを使用）」を学びましたが、ピリオドを使わずに `*` をそのまま用いると、数学的な「行列の積」として計算されます。

```matlab
C = [1 2; 3 4];
D = [5 6; 7 8];

% 行列の積 (線形代数における行列の掛け算)
E = C * D;

% 行列の転置（行と列を入れ替える）
% 変数の後ろにシングルクォーテーション ' をつけます
C_trans = C';

% 逆行列の計算
% inv() は逆行列 (Inverse matrix) を求める関数です
C_inv = inv(C);
```

### 関数の解説
* `inv(行列)` : 指定した正方行列の逆行列を計算して返します。制御系の設計などで頻繁に登場します。
* `'` (転置演算子) : 行列の行と列を反転させます。例えば、行ベクトルを列ベクトルに変換する際にも使われます。

## 3. ターミナルとNeovimを用いた開発
MATLABのGUIエディタを使わず、ターミナル（CUI環境）上でコーディングと実行を完結させることも可能です。ファイルの編集には、NanoではなくNeovimを使用すると効率的です。

1. **ファイルの編集**
   ターミナル上で以下のように打ち込みます。
   `nvim my_matrix.m`
   *ターミナルコマンドの解説*: `nvim` は高機能なテキストエディタであるNeovimを起動し、指定したファイルを開く（存在しなければ新規作成する）コマンドです。

2. **スクリプトの実行**
   ターミナルから直接MATLABを実行するには、以下のコマンドを使います。
   `matlab -batch "my_matrix"`
   *ターミナルコマンドの解説*: `matlab` はMATLABエンジンを呼び出すコマンドです。`-batch` オプションを付けることで、「GUI画面を立ち上げずに、指定したスクリプト（`my_matrix.m`）をバックグラウンドで実行し、終わったら自動的に終了する」という一連の処理をターミナル上だけで完結させることができます。

## 4. 連立方程式の解法（バックスラッシュ演算子）
線形代数において、連立方程式は $Ax=b$ （$A$ は係数行列、$b$ は定数ベクトル、$x$ は未知の変数ベクトル）という形で表されます。
数学的には $x=A^{-1}b$ として解を求めますが、MATLABでは `inv(A) * b` とするよりも、バックスラッシュ `\` を使う方が計算速度が速く、数値的にも安定しています。

```matlab
% 2x + 3y = 8
% 5x - y = 3
% という連立方程式を解く場合

% 係数行列 A
A = [2, 3; 5, -1];

% 定数列ベクトル b
b = [8; 3];

% バックスラッシュ演算子で未知数ベクトル x を求める
x = A \ b;
```
* `\` (バックスラッシュ / 左除算演算子) : $Ax=b$ の方程式を解き、$x$ を返します。MATLAB内で最も高度に最適化されている演算子の一つです。

## 例題：ドローンのローター推力計算
あるドローンに搭載された2つのモーター（モーター1とモーター2）が発揮する推力 $T_1$ [N] と $T_2$ [N] について、以下の連立方程式が成り立つとします。

合計推力の式: $T_1+T_2=10$
トルクのバランス式: $2T_1-T_2=5$

これを行列の形 $AT=b$ で表し、バックスラッシュ演算子 `\` を用いて推力ベクトル $T$ (要素は $T_1$ と $T_2$) を求めるMATLABコードを作成してください。

## 例題の解答

```matlab
% 係数行列 A の定義
% 1つ目の式 (1*T1 + 1*T2 = 10) と 2つ目の式 (2*T1 - 1*T2 = 5) の係数を抜き出します
A = [1, 1; 2, -1];

% 定数ベクトル b の定義
% 右辺の値を列ベクトル (縦長) として定義します
b = [10; 5];

% 連立方程式 AT = b を解き、推力ベクトル T を求める
T = A \ b;

% 結果の出力
disp('各モーターの推力 [T1; T2] は以下の通りです [N]:');
disp(T);
```

### 解答コードの解説
* `A = [1, 1; 2, -1];` : 数式から係数だけを抽出し、2行2列の行列を作成しています。
* `b = [10; 5];` : 右辺の値をセミコロンで区切り、列ベクトルにしています。行列計算で $Ax=b$ を解く場合、$x$ と $b$ は列ベクトルである必要があります。
* `T = A \ b;` : `A` の逆行列を求めて `b` に左から掛ける操作を一撃で行っています。この演算子を使いこなすことで、複雑な制御系や連立微分方程式の解析コードを非常にシンプルに記述できるようになります。
